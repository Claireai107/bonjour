# 본주르 내비게이션·리포트·첫 진입 흐름 개선 설계

## 목표

본주르의 주요 화면에서 하단 탭바를 항상 같은 위치에 고정하고, 첫 가입 이후 사용자가 홈에서 직접 AI 뼈 건강 분석을 시작하도록 흐름을 단순화한다. 반복 분석 결과는 사용자별 이력으로 보존해 리포트 화면에서 날짜별로 조회할 수 있게 하고, 마이페이지의 사용자 선택 기능은 상단 우측으로 이동한다.

## 범위

### 하단 탭바

- 탭바는 76px 높이의 별도 고정 영역으로 유지한다.
- 화면 본문만 스크롤되고 탭바는 스크롤되지 않는다.
- 탭바 예외 화면은 스플래시(`/`), 회원가입(`/signup`), 설문(`/survey`)이다.
- 홈, AI 루틴, 우리동네, 리포트, 마이페이지의 기존 탭바를 같은 고정 규칙으로 통일한다.
- 답변방식 선택, 건강검진 입력, AI 분석, 행동 변화 시뮬레이터, 관심 건강 관리, 새 사용자 추가 화면에도 탭바를 추가한다.
- 보조 화면의 활성 탭은 다음처럼 처리한다.
  - 답변방식 선택, 건강검진 입력, AI 분석: 홈
  - 행동 변화 시뮬레이터: 리포트
  - 관심 건강 관리, 새 사용자 추가: 마이

### 첫 진입 및 재측정 흐름

1. 스플래시 영상이 끝나면 기존 소개 화면을 표시한다.
2. `시작하기`를 누르면 회원가입 화면으로 이동한다.
3. 회원가입 완료 후 홈으로 이동한다.
4. 분석 결과가 없는 홈에는 `AI 뼈건강 분석 시작` 버튼을 표시한다.
5. 이 버튼을 누르면 기존 진단 입력값을 초기화하고 설문 1번으로 바로 이동한다.
6. 설문 첫 문항에서 뒤로 가면 홈으로 돌아간다.
7. 설문 완료 이후의 건강검진 입력 → AI 분석 → 리포트 흐름은 유지한다.
8. 재측정 시 과거 리포트 이력은 삭제하지 않는다.

답변방식 선택 화면은 기능 자체는 유지하지만, 첫 가입 후 기본 분석 경로에서는 건너뛴다. 기본 답변 방식은 현재 저장소 기본값인 손 입력을 사용한다.

### 리포트 이력

- 분석 완료 때마다 활성 사용자 프로필의 리포트 이력에 ISO 날짜와 분석 결과를 추가한다.
- 기존 정책대로 사용자별 최근 20건까지만 유지한다.
- 리포트 우측 상단에 날짜 드롭다운을 표시한다.
- 목록은 최신순이며 기본 선택값은 최신 리포트다.
- 과거 항목을 선택하면 점수, 등급, 위험·보호 요인, 개선 가능성 등 리포트 결과 전체를 선택한 결과로 교체한다.
- 사용자 전환 또는 새 분석 이력 추가 시 선택값을 최신 리포트로 되돌린다.
- 이력은 현재 Zustand 세션 저장 방식을 유지한다. 브라우저 세션을 넘어선 영구 저장은 이번 범위가 아니다.

### 마이페이지 사용자 선택

- 상단 첫 줄에 `마이페이지` 제목을 왼쪽에 둔다.
- 현재 사용자 선택 버튼을 같은 줄 오른쪽에 둔다.
- 선택 버튼에는 현재 사용자 이름, 프로필 아이콘, 펼침 표시를 유지한다.
- 버튼을 누르면 기존 사용자 전환 시트를 그대로 연다.
- 안내 문구는 제목 행 아래에 유지한다.

## 구현 구조

- 기존 각 화면의 `h-dvh`, 세로 flex, 본문 `flex-1 overflow-y-auto`, 마지막 `TabBar` 패턴을 유지한다.
- 공용 `TabBar`는 자체적으로 높이가 줄어들지 않는 하단 영역이 되며, 경로별 활성 탭 매핑을 담당한다.
- 전역 레이아웃 자동 삽입 방식은 사용하지 않는다. 기존 화면의 높이와 고정 CTA에 미치는 영향을 줄이기 위해 필요한 화면에 명시적으로 탭바를 배치한다.
- 고정 CTA가 있는 화면은 스크롤 본문, CTA, 탭바 순서로 배치해 서로 가리지 않게 한다.

## 예외 및 오류 처리

- 저장된 최신 결과가 없는 리포트·루틴·시뮬레이터 직접 진입 방어는 기존 리다이렉트 동작을 유지한다.
- 날짜가 없는 이전 단일 결과는 `현재 리포트`로 표시한다.
- 리포트 이력 인덱스가 변경으로 범위를 벗어나면 최신 유효 항목을 사용한다.
- 탭바는 중복 렌더링하지 않으며 스크롤 콘텐츠와 고정 CTA를 가리지 않아야 한다.

## 검증 기준

- 프로덕션 빌드와 TypeScript 검사가 성공한다.
- 스플래시, 회원가입, 설문에는 탭바가 없고 그 외 지정 화면에는 탭바가 한 번만 표시된다.
- 긴 콘텐츠를 스크롤해도 탭바 위치가 바뀌지 않는다.
- 스플래시 → 회원가입 → 홈 → 설문 1번 흐름이 동작한다.
- 재측정 시작 시 현재 설문·검진·결과는 초기화되지만 이전 리포트 이력은 남는다.
- 리포트 기본값은 최신이며 과거 날짜 선택 시 화면 결과가 변경된다.
- 사용자 전환 시 해당 사용자의 최신 리포트가 선택된다.
- 마이페이지 사용자 선택 버튼이 제목 오른쪽에 있고 기존 전환 시트를 연다.

## 제외 범위

- 회원 인증 또는 서버 계정 저장
- 리포트 이력의 서버·로컬 영구 저장
- 답변방식 선택 기능의 재설계
- 설문 문항, 예측 모델, 처방 규칙 변경
- 전체 화면을 새 공용 레이아웃으로 전환하는 대규모 리팩터링
